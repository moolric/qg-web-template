Getting started

The prefered method for using this template is to use the Web Project Jumpstarter from Git, which uses Node.js, NPM, and Gulp. This allows you to build both Franchise websites and Agency templates while using our best practice development methods.

  1. Ensure you have a Git client and Node.js installed
  2. Fork the Web Project Jumpstart repo:
    $ git clone https://github.com/qld-gov-au/jumpstart
    For more information see how to fork a github repo
  3. Set up your project:
    $ npm init
  4. Check out the readme that comes with the Web Project Jumpstarter
  5. Read the documentation here to understand how the template works

Usage

There are two main ways to use the Web Template:

  1. CDN Template: To build franchise websites
  2. Local Template: To customise a new website (or template), or host a local franchise website - for Agencies, campaign sites etc.

Technology

To help with rapid development, the web template is now tightly integrated with Bootstrap 3. It also includes jQuery and a suite of custom Javascript features.

Compliance

Queensland Government Web Template (web template) is build to comply with the IS26 Guidelines.
