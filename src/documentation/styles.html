The QG Web Template follows Bootstrap closely in implementation style. The major exception to this is the current implementation of forms.

Below is a list of implemented Bootstrap styles. See the bootstrap website on css for further advice.

Typography

Queensland Government websites should use the Lato web font. This is included by default in the Queensland Government Web Template. In most instances when using the template, you should not have to worry about custom typography.

For more information on implementation, see the bootstrap typography section.
