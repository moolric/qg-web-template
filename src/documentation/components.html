Dropdowns

Dropdowns are only to be used for web applications, not forms.

Due to the nature of the Forms system being used, dropdowns are not advised for web forms.

Unlike Bootstrap, the Web Template automatically provides a caret on dropdowns. See custom icon to disable this feature.

For more information on implementation, see the bootstrap dropdowns section.

Carer Business Discount Card

If you receive a Centrelink carer payment or carers allowance and live in Queensland, you are eligible for a Carer Business Discount Card,which recognises the contribution carers make to the community and provides discounts on products and services.
